# Xsec-dt

This repository contains the complete Digital Twin abstraction and experimental framework used for security and risk analysis research.

The provided ZIP archive includes all required Python source files, configuration files, and supporting data needed to reproduce the experiments.

## Contents

The repository includes the following key files:

- component_registration.py  
- dependency_graph_builder.py  
- graph_utils.py  
- cost_benefit.py  
- decision_rules.py  
- encoding.py  
- feature_extraction.py  
- interaction_analysis.py  
- normalization.py  
- patch_ranking.py  
- risk_scoring.py  
- run_experiment.py  
- shap_computation.py  
- shap_global.py  
- shap_local.py  
- manifest_example.json  

## Description

The code implements a publication-safe Digital Twin abstraction for modeling system components, dependencies, and risk propagation.  
It supports feature extraction, normalization, risk scoring, patch ranking, decision rules, and explainability analysis using SHAP.

All data, Python files, and necessary configuration files are included in the ZIP archive.
